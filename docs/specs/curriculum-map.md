# Curriculum Map — Greek Learning App

**Статус:** Draft v2 (после валидации)
**Уровни:** A1 (6 юнитов), A2 (9 юнитов), B1 (8 юнитов)

Состав каждого юнита: 18–22 карточки + 4–6 упражнений типов 1–4, 6, 7 + мини-тест.
**Принцип A2+:** больше фраз, меньше изолированных слов.

---

### A2-3: Деньги и оплата — Χρήματα και πληρωμές

## Примечания для batch-пайплайна

**Дубли исправлены:**
- `η απόδειξη` → только A2-1 (A2-3 убрана)
- `ο λογαριασμός` → только A2-4, из A1-5 убран
- `παραγγέλνω` → только A2-4, из A1-5 убран
- A1-5 теперь только лексика продуктов питания

**Контекстные замены:**
- `κάθομαι στον τρίτο όροφο` → `μένω στον τρίτο όροφο` (A1-4, κάθομαι = сидеть, не жить)
- `αναζητώ` → `ψάχνω` (A2-5, разговорное)
- `μετρό` → `ταξί` (A2-7, Кипр-релевантно)
- `ο σταθμός` → `ο σταθμός λεωφορείων` (A2-7, уточнение)
- `θερμοκρασία` → `πυρετός` + `βήχας` + `κρυολόγημα` (A2-8)
- `η τιμή με ΦΠΑ`, `χαρτονόμισμα` → убраны из A2-3
- `μπορώ να πληρώσω σε δόσεις` → `δεν έχω μετρητά` (A2-3, рассрочка = B1)
- `η συνταγή` в A2-4 → `η κράτηση` (бронь столика)
- `ψάχνω για δυάρι` → `ψάχνω για διαμέρισμα με δύο δωμάτια` (A2-9, универсальнее)

**Грамматический фокус добавлен** ко всем 23 юнитам (A1/A2/B1).

**Для `generate_content.py`:**
- `unit_topic` = название юнита (например: `Αγορές`)
- `vocabulary_list` = слова/фразы из колонки "Слово / Фраза"
- `level` = A1 / A2 / B1
- `competence` = строка компетенции юнита
