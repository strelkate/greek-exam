# Greek Learning App — Design Spec
**Date:** 2026-04-06
**Scope:** Full app design including architecture, exercise types, UI direction

---

## 1. Общая архитектура

### Стек
- **Frontend:** Telegram Mini App (React + Vite)
- **Backend:** FastAPI (Python)
- **Database:** PostgreSQL
- **Content generation:** офлайн-скрипты через Claude API → PostgreSQL

```
Telegram Client
    └── Mini App (React)
            └── FastAPI (Python)
                    └── PostgreSQL
                          ├── users (telegram_id, прогресс, streak)
                          ├── curriculum (юниты, темы, уровни)
                          ├── exercises (готовые задания)
                          └── user_progress (результаты, spaced rep)
```

Контент генерируется один раз офлайн, приложение только отдаёт его.

### Разделы приложения — 2 раздела

| Раздел | Описание |
|--------|----------|
| **Учёба** | Юниты по темам и уровням A1/A2/B1 |
| **Словарь** | Spaced repetition карточки |

Раздел "Экзамен" **отсутствует**. Подготовка к экзамену встроена органически: все упражнения построены на реальных экзаменационных форматах A2 (ΠΙΣΤΟΠΟΙΗΣΗ ΕΛΛΗΝΟΜΑΘΕΙΑΣ).

---

## 2. Curriculum (структура контента)

### Уровни и юниты

```
A1 (6-8 юнитов)
  ├── Приветствия и знакомство
  ├── Числа, дни, время
  ├── Семья
  ├── Еда и напитки
  └── ...

A2 (8-10 юнитов) ← на основе реальных экзаменационных билетов
  ├── Покупки и магазины
  ├── Транспорт и маршруты
  ├── Работа и профессии
  ├── Здоровье
  └── ...

B1 (8-10 юнитов)
  ├── Путешествия
  ├── Культура и традиции
  └── ...
```

### Состав юнита
- 15–20 карточек словаря (spaced repetition)
- 4–6 упражнений разных типов
- Мини-тест по итогам юнита

---

## 3. Типы упражнений — 7 типов

Все типы повторяют реальные форматы экзамена A2. Пользователь учит язык и одновременно привыкает к формату.

### Тип 1: ΣΩΣΤΟ / ΛΑΘΟΣ
Прочитай текст, отметь утверждения как верные (ΣΩΣΤΟ) или неверные (ΛΑΘΟΣ).

### Тип 2: Сопоставление
Перетащи или тапни: соедини начало фразы с правильным окончанием. Drag & drop или tap-to-match.

### Тип 3: Выбор из вариантов
Выбери правильный ответ из 3–4 вариантов (α/β/γ/δ).

### Тип 4: Заполни пропуск
Выбери слово из банка слов, чтобы заполнить пропуски в тексте. Слов в банке чуть больше нужного.

### Тип 5: Карточки словаря
Греческое слово ↔ перевод. Алгоритм spaced repetition (по результатам: знал/не знал).

### Тип 6: Περιγραφή εικόνας — Описание → Изображение *(новый)*
Источник: формат Ερώτημα 1 из экзамена ΚΑΤΑΝΟΗΣΗ ΠΡΟΦΟΡΙΚΟΥ ΛΟΓΟΥ (упрощённая версия без аудио).

**Механика:**
- Текст с описанием человека/предмета/места показывается вверху
- Ниже — сетка из 6–8 изображений (2–3 колонки)
- Пользователь тапает правильное изображение
- Тап → подсветка выбора → кнопка "Подтвердить"
- После ответа: зелёная/красная рамка, правильное изображение подсвечено
- Использованные изображения становятся серыми (каждое подходит только раз)
- Прогресс: счётчик вверху (2/6)
- Кнопка "Далее" → следующее описание

**Контент:** 6–8 описаний на одно упражнение, 1–2 лишних изображения без описания.

### Тип 7: Διάλογος — Диалог → Форма *(новый)*
Источник: формат Ερώτημα 4 из экзамена ΚΑΤΑΝΟΗΣΗ ΠΡΟΦΟΡΙΚΟΥ ΛΟΓΟΥ (упрощённая версия без аудио).

**Механика:**
- Диалог показан вверху, свёрнут до 3–4 строк, есть кнопка "Показать полностью"
- Реплики визуально разделены по участникам (имя + текст)
- Ниже — таблица с полями для заполнения (5–7 строк)
- Банк слов внизу экрана
- Тап на слово из банка → заполняет активное поле
- Тап на заполненное поле → возвращает слово в банк
- "Проверить" → каждое поле зелёное/красное + показывается правильный ответ
- Слов в банке на 1–2 больше нужного

**Контент:** диалог 10–15 реплик, 5–7 полей в таблице.

## 4. Глобальная настройка: Перевод заданий

Переключатель в настройках приложения (не в каждом упражнении).

| Состояние | Отображение инструкции |
|-----------|----------------------|
| **Вкл** (по умолчанию) | Греческий + перевод в скобках: *"Σημειώνετε ΣΩΣΤΟ ή ΛΑΘΟΣ (отметьте верно/неверно)"* |
| **Выкл** | Только греческий — реальный экзаменационный формат |

Дополнительно: упражнения внутри юнитов постепенно **обучают читать инструкции** на греческом. Новые экзаменационные формулировки вводятся как часть учёбы, не забрасывают пользователя в глубину сразу.

---

## 5. Геймификация (лёгкая, Duolingo-стиль)

Принцип: интерактив есть, давления нет.

| Элемент | Описание |
|---------|----------|
| Счётчик ответов | Итоговый экран после упражнения: X/Y правильных |
| Streak | Количество дней подряд с активностью |
| Прогресс юнита | Сколько упражнений пройдено из N |

**Отсутствуют:** XP, уровни, лиги, таймеры, жизни, штрафы.

---

## 6. Визуальный стиль

- Направление: **Duolingo-подобный** — яркие акцентные цвета, скруглённые элементы, крупные тапабельные карточки, чёткие состояния (верно/неверно/нейтрально)
- Зелёный для правильных ответов, красный для неправильных — стандартная палитра
- Минималистичный UI без перегруза, фокус на контенте упражнения

---

## 7. Генерация контента

Контент создаётся офлайн через скрипты с Claude API:
- Тексты описаний для Типа 6 (Περιγραφή εικόνας)
- Диалоги и таблицы для Типа 7 (Διάλογος)
- Тексты и вопросы для всех остальных типов
- Словарные карточки для юнитов

Скрипты запускаются вручную по мере необходимости. Приложение только отдаёт готовый контент из БД.
